# 📦 Inventory Management System (C++ OOP Project)

A console-based Inventory Management System built using **C++ and Object-Oriented Programming (OOP)** principles.  
This project manages products, suppliers, sales, and file-based data storage in a structured and modular way.

---

## 🚀 Features

- ➕ Add new products
- ✏️ Edit product details
- ❌ Delete products
- 👤 Manage supplier information
- 💾 File handling (data persistence)
- 📊 Generate basic sales and stock reports
- 🔍 View all inventory data in structured format

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
  - Classes & Objects
  - Inheritance
  - Encapsulation
  - Polymorphism
- File Handling in C++
- Vectors and STL
- Modular Programming
- Exception Handling (if used)
